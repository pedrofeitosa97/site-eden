# Éden — Site Oficial

Site oficial do projeto **Éden**, um servidor de FiveM roleplay com temática sobrenatural em Los Santos.

## Funcionalidades

- **Apresentação** — Landing page com informações do projeto
- **Lore** — História principal e lores pessoais com navegação por categorias
- **Loja** — Catálogo de produtos sincronizado com o servidor
- **Allowlist (Quiz)** — Formulário automatizado com timer de 4 minutos por pergunta e 3 tentativas
- **Links** — Redes sociais e Discord da comunidade

## Tecnologias

- [Next.js 14](https://nextjs.org/) — Framework React
- [Tailwind CSS](https://tailwindcss.com/) — Estilização
- [Framer Motion](https://www.framer.com/motion/) — Animações
- [Lucide React](https://lucide.dev/) — Ícones

## Como executar

```bash
# Instalar dependências
npm install

# Modo desenvolvimento
npm run dev

# Build de produção
npm run build
npm start
```

O site estará disponível em `http://localhost:3000`.

## Design

Interface com estética holográfica escura, utilizando tons de rosa pastel, roxo e preto para criar uma atmosfera misteriosa alinhada com a lore do servidor.
